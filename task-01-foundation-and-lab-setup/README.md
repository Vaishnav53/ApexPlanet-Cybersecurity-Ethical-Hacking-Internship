# Task 1 – Foundation & Environment Setup

## Objective

Build strong fundamentals in cybersecurity, networking, cryptography, Linux administration, and establish a professional virtual penetration testing laboratory.

---

## Learning Outcomes

* Understand cybersecurity principles
* Learn CIA Triad concepts
* Study common cyber threats
* Configure a cybersecurity lab
* Learn Linux fundamentals
* Understand networking concepts
* Learn cryptography basics
* Gain familiarity with industry tools

---

## Task Components

### 1. Cybersecurity Basics

* CIA Triad
* Threat Types
* Attack Vectors

### 2. Lab Environment Setup

* VirtualBox / VMware
* Kali Linux
* Metasploitable2
* Host-Only Network Configuration

### 3. Linux Fundamentals

* File System Navigation
* User Permissions
* Package Management
* Networking Commands

### 4. Networking Basics

* OSI Model
* TCP/IP
* DNS
* HTTP/HTTPS
* IP Addressing
* Subnetting
* NAT

### 5. Cryptography

* Symmetric Encryption
* Asymmetric Encryption
* Hashing
* SSL/TLS
* OpenSSL Demonstration

### 6. Tool Familiarization

* Wireshark
* Nmap
* Burp Suite
* Netcat
