# Executive Summary

The objective of Task 1 was to establish a secure cybersecurity laboratory environment and develop foundational knowledge in cybersecurity, networking, Linux administration, cryptography, and security tools.

The lab was configured using Kali Linux as the attacker machine and Metasploitable2 as the target machine in an isolated Host-Only virtual network.

---

# Objectives

* Understand cybersecurity fundamentals.
* Study the CIA Triad.
* Learn common cyber threats and attack vectors.
* Configure a virtual cybersecurity lab.
* Learn Linux fundamentals.
* Understand networking concepts.
* Explore cryptography principles.
* Gain familiarity with common security tools.
