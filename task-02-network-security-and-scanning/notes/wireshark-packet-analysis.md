# Wireshark Packet Analysis

## Introduction

Wireshark is a network protocol analyzer used to capture and inspect network traffic in real time.

It is one of the most widely used tools for troubleshooting, security analysis, and protocol learning.

---

# Features

- Live Capture
- Packet Filtering
- Protocol Analysis
- Stream Following
- Export Packets
